# FPGA Basic Projects

Collection of FPGA projects implemented in Verilog HDL on the Nexys A7-100T FPGA board.

## Projects

### 1. Simple Signed ALU
- Signed arithmetic
- Seven-segment display multiplexing
- Overflow detection

### 2. Future Projects
- Counters
- UART
- SPI
- VGA
- FSM designs

## Tools Used
- Vivado 2025.2
- Verilog HDL

## FPGA Board
- Digilent Nexys A7-100T
